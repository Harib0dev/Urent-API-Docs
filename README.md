# Urent-API-Docs
Неофициальная документация API сервиса аренды самокатов Юрент, основанная на анализе взаимодействия мобильного клиента с сервером с помощью реверс-инженеринга и инструмента для анализа сетевого трафика Loon.

⚠️ Данный проект создан исключительно в исследовательских и образовательных целях.
